# server

### Dependencies
1. Python3
2. python-pip
3. python-virtualenv

### Full usage example

1. Create an ENV in the root directory (same one the README is in)  and activate it.
```
python -m venv ENV
source ENV/bin/activate
```

3. Install requirements.
```
pip install -r requirements.txt
```

3. Run
```
flask run

```
